# schonherz-doom
A mod to make Doom easier to view on Schönherz Mátrix.

# Features of this mod:
- Make rooms brighter
- Pistol can be used to zoom out


# GZDoom Display settings used during gameplay:
- Gamma correction: 1.45
- Brightness: 0.05
- Contrast: 1.0
- Saturation: 2.75

# Resolume Arena settings:
- 1920x1080@25F
- Position x: 0
- Position y: -56
- Scale: 94%
